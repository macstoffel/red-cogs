# Quiz Cog

Een quiz cog voor Discord.

## Installatie

Om deze cog te installeren, voer de volgende commando's uit:

1. `pip install redbot`
2. `git clone https://github.com/jouw-naam/quiz-cog.git`
3. `cd quiz-cog`
4. `pip install -r requirements.txt`

## Configuratie

Om deze cog te configureren, wijzig de bestanden in de `config` map.

## Gebruik

Om deze cog te gebruiken, voer de volgende commando's uit:

1. `/quiz start` om een nieuwe quiz te beginnen
2. `/quiz score totaal` om de totale score van alle leden te zien
3. `/quiz @member` om de score van een ander lid te zien
